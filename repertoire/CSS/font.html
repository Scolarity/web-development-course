For sans-serif and serif you need to look at w3school, serif has more texture and added things from sans-serif, sans-serif is simpler

Cursive and Fantasy are two font families that you almost never use because they are difficult to read.

You could use monospace for code editors

For most browsers, the default font in the font-family serif, is arial, but verdana, for example is also a good one.

You can specify the font in the font-family already, just have it be in front of the family name: 'font-family: verdana, sans-serif;'.

If the browser doesn't support that font type though, it will show their sans-serif default.

To combat this, there are web safe font, that are supported by most web browsers.

W3schools CSS Web Safe Fonts Page

No font is 100% compatible though, if you look at this website for example: cssfontstack, and then you would need fallbacks written after the first font: 'font-family: verdana, Arial (the fallback), sans-serif;'.

You can find new fonts at the google fonts page, and you can find a link tag there, add that to your css file with import [link] or at the head of a page.

Then you can just put them in your font-family line in your code with the appropriate name of the font

If you get it from there everyone will see it if they have an internet connection, they don't need their browser to support it.
