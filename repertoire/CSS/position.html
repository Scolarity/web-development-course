Position CSS

If the position is static it is how it would normally be in plain HTML:

If the position is relative it is static, but it doesn't affect other elements and it can be positioned by using left, right, top, bottom expressions in css.
They add margins left, right, ect. respective to their static position. It does leave a "ghost" element behind where the static relative position was.

If the position is absolute it is relative but it is relative to its parent element. This does not leave a ghost element behind, so it can affect other elements, just not how you expect.

A fixed position is like an absolute position but it is always relative to the screen, so it stays where it is on the screen even if you scroll down
This can be useful when you are create a navigation bar for example.
